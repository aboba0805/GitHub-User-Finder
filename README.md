# Описание программы

**GitHub User Finder** — это GUI-приложение для поиска пользователей GitHub с использованием GitHub API.

Программа позволяет:

* искать пользователей GitHub;
* отображать результаты поиска;
* добавлять пользователей в избранное;
* сохранять избранных пользователей в JSON;
* автоматически загружать избранное при запуске программы;
* открывать GitHub-профили в браузере.

## Возможности

* Поиск пользователей GitHub через API
* Отображение результатов поиска
* Добавление пользователей в избранное
* Сохранение избранного в JSON
* Загрузка избранного из JSON
* Открытие профиля GitHub в браузере
* Проверка корректности ввода
* Защита от дублирования записей
* Обработка ошибок подключения

# Инструкция по запуску

## Требования

* Python 3.8 или выше
* Интернет-соединение
* Библиотека requests

## Установка

1. Откройте терминал
2. Перейдите в папку проекта:

```bash
cd github-user-finder
```

3. Установите зависимости:

```bash
pip install -r requirements.txt
```

---

# Запуск программы

```bash
python main.py
```

# Использование GitHub API

Программа использует GitHub API для поиска пользователей.

Пример запроса:

```text
https://api.github.com/search/users?q=octocat
```

API возвращает данные в формате JSON, которые отображаются в интерфейсе приложения.

---

# Примеры использования

## Поиск пользователя

1. Введите имя пользователя GitHub
2. Нажмите кнопку `Search`
3. Результаты появятся в списке

Примеры запросов:

* octocat
* torvalds
* microsoft
* google
* python

---

## Добавление в избранное

1. Выполните поиск
2. Выберите пользователя из списка
3. Нажмите кнопку `Add to Favorites`

Пользователь сохранится в файл `favorites.json`.

---

## Открытие профиля

1. Выберите пользователя
2. Нажмите кнопку `Open GitHub Profile`

Профиль откроется в браузере.

---

# Валидация данных

Программа проверяет:

* поле поиска не должно быть пустым;
* пользователь должен быть выбран из списка;
* нельзя добавить одного пользователя дважды;
* ошибки подключения к GitHub API.
